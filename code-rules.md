# C 语言格式规范

## 代码风格

- 相同类型变量定义尽量在同一行，如 `uint32_t a, b, c`
- 函数体内变量定义在函数体开始处，变量定义完后接一个空行
- 类型必须写完整：`unsigned int`（禁止裸 `unsigned`）
- 指针星号两侧各 1 个空格：`char * const bar`
- 单行 if 后接单语句必须带大括号：`if (x) { return; }`
- goto 仅用于错误返回：goto 到函数末尾统一清理并 return
- 遵循 MISRA C 语言标准

## 注释

- 使用英文，简洁
- 函数、变量功能容易从代码中感知时不需要注释

## 打印

- 打印简洁
- 功能测试中报错不打印，但要求给出返回值
- 功能测试结果给出打印；测试失败时打印返回值
