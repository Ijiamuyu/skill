# 约会规则

> 首次线下见面及之后的真实相处。线上聊天见 [`chat-rules.md`](./chat-rules.md)。具体偏好以该案例档案为准。

## 核心

**状态 > 内容**。默认把对方当朋友，**别带"找对象"心态**。

## 六要诀

1. **自信** —— 慕强是人性。**和平常一样，放松是关键**。
2. **真诚** —— "我也是第一次约，有点紧张"是示好不是示弱。
3. **大气** —— 吃饭、买水、坐车不抠门；不显摆。
4. **态度** —— 她不满时：**先共情 → 再处理 → 永远最后讲理**。
5. **话题** —— **糗事 > 成就**，自嘲拉近距离。**幽默是王炸**，但≠耍贫嘴。
6. **形象** —— 干净整洁是基础：头发有造型、无异味、指甲净、鞋不邋遢。**合身 > 贵**。

## 决策权

大事自己做主（场景、路线、节奏、点菜、收尾）；小事征求她（口味、忌口、是否想去某处）。什么都问"你想去哪" = 无趣 / 没担当。

## 别踩的坑

- 时间地点啰嗦铺垫
- 临场核对清单式动作（里侧 / 买水 / 买单 / 送）
- 告别油腻或承诺下次
- 到家长篇感慨
- 24h 内主动提下次

## 红旗（触发则评估）

- 全程要你买单且金额偏高（试探）
- 抱怨前任 / 全世界（受害者心态）
- 迟到不解释 / 失联 / 说一套做一套（不可靠）
- 控制你穿衣 / 交友 / 行为
- PUA 痕迹
