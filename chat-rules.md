# 找对象聊天规则

## 原则

0. **先当朋友** —— 不要带着"找对象"心态去聊。
1. **轻松明快** —— 避免沉重、套路、刻意讨好。
2. **以分享带动分享** —— 自己先抛感受，带动她也分享。
3. **兴趣交集切入** —— 不凭空找话题。
4. **镜子原则** —— 希望对方展示什么，就先展示什么。
5. **轻量节奏** —— 不一次聊透，留余地。
6. **不查户口** —— 拒绝盘问式信息采集。
7. **场景匹配** —— 尊重对方的沟通习惯。

## 阶段

```
破冰（兴趣切入，留钩子）
  ↓ 几次轻量聊天
建立熟悉感（同好分享 + 日常关心）
  ↓ 双向主动 + 线下相关信号
引向线下（具体方案）
  ↓ 首次见面通过
建立真实相处
  ↓ 三原则评估
决策
```

## 展示点

让作品替你说话，她主动看一眼胜过你讲十句。

- **B 站（甲木俞）**：旅行 vlog，触发话题时顺势一句："B 站搜'甲木俞'能看到"
- **知乎**：观点犀利，**不主动给陌生人**，留给同频者

## 边界

- **不强求**：婉拒就退回当前阶段
- **红旗**：功利试探、受害者心态、不可靠、控制欲、套路感 → 退出
