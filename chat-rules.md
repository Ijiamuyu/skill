# 找对象聊天规则

## 原则

1. **以分享和好奇为主** —— 听她说、了解她，不是展示自己
2. **以兴趣交集为切入口** —— 不凭空找话题
3. **真实好奇胜过套路** —— 只在真正感兴趣的领域展开
4. **镜子原则** —— 希望对方展示什么，就先展示什么
5. **轻量节奏** —— 不一次聊透，留余地
6. **不查户口** —— 拒绝盘问式信息采集
7. **场景匹配** —— 尊重对方的沟通习惯

## 阶段

```
破冰（兴趣切入，留钩子）
  ↓ 几次轻量聊天
建立熟悉感（同好分享 + 日常关心）
  ↓ 双向主动 + 线下相关信号
引向线下（具体方案）
  ↓ 首次见面通过
建立真实相处
  ↓ 三原则评估
决策
```

## 边界

- **不强求**：婉拒就退回当前阶段
- **红旗**：功利试探、受害者心态、不可靠、控制欲、套路感 → 退出
