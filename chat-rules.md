# 聊天

接触全过程的对话与相处方法论，覆盖线上聊天与线下见面。

## 核心原则

1. **以兴趣交集为切入口** —— 不凭空找话题
2. **真实好奇胜过套路** —— 只在自己真正感兴趣的领域展开
3. **镜子原则** —— 希望对方展示什么，就先展示什么
4. **轻量节奏** —— 不一次聊透，留余地
5. **不查户口** —— 拒绝盘问式信息采集
6. **场景匹配** —— 尊重对方的沟通习惯（线上 vs 线下、话题深度）

## 阶段流程

```
线上：破冰（兴趣切入，留钩子）
  ↓ 几次轻量聊天
线上：建立熟悉感（同好分享 + 日常关心）
  ↓ 出现双向主动 + 线下相关信号
引向线下（具体方案，非开放式提问）
  ↓ 首次见面通过
线下：建立真实相处（场景匹配 + 节奏自然）
  ↓ 三原则持续评估
决策
```

## 关键边界

- **不强求**：婉拒就退回当前阶段，不强行推进
- **主动 vs 回应 ≈ 6:4**：完全被动或完全主动都不健康
- **首次线下**：低成本低压力，结合场景偏好（季节、户外/室内）
- **线上 vs 线下**：线上可观察分寸感、文字态度；线下才能观察真实气场、肢体语言、对服务员的态度
- **红旗出现**：功利试探、受害者心态、不可靠、控制欲、套路感 → 果断退出

## 适用范围

- 已建立初步接触（已互加联系方式或已线下相识）
- 贯穿从破冰到决策的全过程
- 三原则评估作为决策依据，不替代三原则本身（见 `README.md`）
