# 找对象聊天规则

## 心态

- **先当朋友**，别带"找对象"心态。轻松明快，不套路不讨好。
- **分寸感**：不越界。对人不能抨击、对事不能干涉、对物不能质疑。
- **识趣**：对方嗯嗯哦哦 + 不反问 → 撤退。她有筛选权，你也有放弃自由。

## 聊法

- **兴趣交集切入**，不凭空找话题。只用自己真实的兴趣去碰对方的兴趣，不编造经历。
- **破冰用安全兴趣**：户外/爬山 —— 人人能聊，无门槛。阅读观点鲜明，熟悉后再聊。
- **询问代替陈述**：多问开放题，少用是非题。把表达机会让给对方。
- **三七法则**：70% 聊她，30% 聊你。你是捧哏。
- **分享带动分享**：自己先抛感受，带动她也分享。
- **轻量节奏**：20-40 分钟收，不一次聊透。黄金时段：20:00-23:00 / 周末上午。

## 展示

让作品替你说话，她主动看一眼胜过你讲十句。

| 类型 | 方式 |
|------|------|
| 硬性条件 | 直接干净，不带感情 |
| 软件价值 | 谦虚低调，欲扬先抑 |
| 共同兴趣 | 推给她，附带点赞 |

**抛饵，不硬塞**：分享一个片段留缺口，她好奇追问再展开。
- B 站（甲木俞）：旅行 vlog，她追问再给。
- 知乎：观点犀利，不主动给陌生人。

## 边界

- 婉拒 → 退回当前阶段，不强求。
- 红旗（功利试探、受害者心态、不可靠、控制欲、套路感）→ 退出。

## 阶段

```
破冰 → 几次轻聊 → 建立熟悉感 → 引向线下 → 真实相处 → 决策
```
